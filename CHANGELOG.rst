I2C library change log
======================

-----
  * Multiple write supported.
  * Correction in r/w in i2c_slave.xc

3.1.2
-----

3.1.1
-----

  * Document reg_read functions more clearly with respect to stop bit
    behavior.


3.0.0
-----
